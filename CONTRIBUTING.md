# Contributing to Biobank Variant Explorer

We welcome contributions to improve the efficiency and flexibility of this tool.

## How to Contribute

1.  **Fork the Repository**: Create your own copy of the project.
2.  **Create a Branch**: `git checkout -b feature/fast-search`
3.  **Make Changes**: Implement your feature or fix.
4.  **Test**: Ensure the script works with dummy .bim files.
5.  **Submit a Pull Request**: Describe your changes and why they are needed.

## Guidelines

*   **Performance**: Since this tool scans thousands of files, efficiency is key. Avoid loading full files into memory.
*   **Compatibility**: Ensure shell commands work on standard Linux distributions (CentOS/Ubuntu).

## Reporting Issues

If you encounter issues with specific file formats or paths, please open an issue with details.
